# Idea 13: "Un Sub Para Cada Hambre"

## Concepto

Mostrar versatilidad de Subway para diferentes ocasiones/momentos del día.

---

## Creativo

**Serie 3 piezas**:

```
HAMBRE DE MEDIODÍA
#7 The MexiCali
Potente, satisfactorio, sin culpa

HAMBRE DE ALMUERZO RÁPIDO
#1 The Philly
5 minutos, sabor garantizado

HAMBRE DE ALGO DIFERENTE
Sub Clásico
Infinitas opciones, tu decisión

Un sub para cada hambre.
Subway.
```

Visual: 3 subs diferentes, 3 momentos, 3 necesidades

---

**Recomendación**: ⭐⭐⭐⭐ (7.5/10)

Versatil, muestra amplitud de ocasiones, relatable.
